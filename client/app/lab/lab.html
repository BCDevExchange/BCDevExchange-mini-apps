Warning

  • The Labs contain work being designed or under construction
  • This work may change, disappear, or be buggy
  • It’s not to be relied on for anything mission critical or even mildly important
  • Hardhats and goggles, please!

The lab is intended to:

  • Be a safe place to experiment, build and share early stage work
  • Provide disclaimers to set appropriate expectations
  • Provide some tools to assist development efforts
  • Act as a gateway to projects under development

Labs

The Labs are a safe place for early adopter public sector members of the BCDevExchange to experiment in the open. They are a reflection of the BCDevExchange’s commitment to open development and co-creation.

In the Labs you will find experiments; ideas that need to be proven. Everything in the Labs needs to be tested, validated and refined. Like any experiments, some will work and others will fail. So be prepared that some of what you see here will disappear.

DevOps Namespace

DevOps namespace allows qualified program owners to
  1. Reserve https://lab-<your_namespace>.bcdevexchange.org
  2. Test drive in the reserved namespace any Docker enabled web application downloadable from a GitHub repository. The service is powered by Jenkins supporting CI.
  3. Alternatively, program owners can also use the namespace to proxy an existing web site.
